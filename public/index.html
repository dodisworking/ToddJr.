Local / API connection

API requests go to: …

Todd Jr. defaults to port 3456 locally (so :3000 can be another app). Run npm start and open the exact URL printed in the terminal. Override with PORT=3000 npm start if you prefer.

Railway API, browser on this Mac: paste your app root https://YOUR-SERVICE.up.railway.app (no /api suffix, no trailing slash) → Save. In Railway → Variables on that service add LOCAL_DEV_CORS=1 and redeploy so localhost is allowed by CORS.

OpenAI key (local — easiest options)

Easiest: in the same folder as package.json, create a file named openai.key with your sk-… on one line, then restart npm start. No browser needed.

Or paste below — only when Todd runs on this Mac (the server sees 127.0.0.1). For Railway, set OPENAI_API_KEY in Variables instead.

Pasted keys live in server memory until you restart Node. They are not written to disk. openai.key and .env are more permanent for local dev.

TextEdit / file: the file must be named .env (with the dot — not .nv), in the same folder as package.json, with one line: OPENAI_API_KEY=sk-your-key-here then restart npm start.
